# Logo de Farmacia

Para agregar el logo a los reportes, coloca un archivo de logo en esta carpeta con uno de estos nombres:
- `logo.png` (recomendado para transparencia)
- `logo.jpg` 
- `logo.jpeg`

El logo debe tener las siguientes características:
- Formato: PNG, JPG o JPEG
- Tamaño recomendado: 200x200 píxeles o mayor
- Fondo transparente (solo PNG)
- Relación de aspecto cuadrada o rectangular

El logo se mostrará en la esquina superior izquierda de todos los reportes PDF con un tamaño de 20x20mm.

Si no hay logo, los reportes funcionarán normalmente sin él.

**Orden de prioridad**: Si hay múltiples archivos, se usará en este orden:
1. logo.png
2. logo.jpg  
3. logo.jpeg
